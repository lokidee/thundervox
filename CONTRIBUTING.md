# Contributing to THUNDER_VOX

Thanks for your interest! This is a small project built for fun.

## Found a bug?

[Open an issue](https://github.com/lokidee/thundervox/issues/new) and describe what happened.
Include what you expected vs what actually happened.

## Want to add something?

1. Fork this repo
2. Make your changes
3. Open a pull request

Keep it simple. Keep it fun. For the Emperor.
